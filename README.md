# IP Geolocation Tool
Python IP geolocation tool with both command-line (CLI) and Flask web interface.

## Features
- IP validation (IPv4/IPv6)
- Real-time geolocation lookup
- CLI and web interfaces

## Usage

### CLI Version
```bash
python cli_version.py
```
